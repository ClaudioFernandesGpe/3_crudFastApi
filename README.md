# CRUD com FasApi

Projeto CRUD com FastApi da Trilha de Python do canal DESVENDANDO O CÓDIGO, do professor Marcos Caldas.
Esse é um projeto básico para aprendizado inicial do FastApi e backend em python.
Neste projeto será utilizado o armazenamento em memória.

- FasApi
- Pydantic

- Swagger
